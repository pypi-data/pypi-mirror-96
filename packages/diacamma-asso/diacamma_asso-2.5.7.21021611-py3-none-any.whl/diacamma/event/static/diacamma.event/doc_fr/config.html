Configurations¶

La configuration du gestionnaire d’évenement est seulement relative à la configuration des grades, diplômes ou niveaux.

Dans le menu Administration/Configuration des diplômes vous pouvez préciser la dénomination de vos diplômes ainsi que saisir hierachiquement les diplômes et éventuellement les sous-diplômes relatifs à votre structures.
