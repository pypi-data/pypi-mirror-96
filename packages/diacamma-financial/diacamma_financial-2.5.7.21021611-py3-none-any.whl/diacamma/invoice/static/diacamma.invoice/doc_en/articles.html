The articles¶

Creating and editing¶

From the menu Financial/Invoice/Articles you can define all of your billable items.

You can add, edit or delete an article. The removal is not possible if the article is used in an invoice.

With each article, you must define an accounting code imputation for automatic entries generation.

In addition, if you realize VAT invoices, you must specify for each items, the tax rate to apply.
